# Starting Horse For Streamers and Busy People - version 3.0.1.1

## Overview
**Starting Horse For Streamers and Busy People** lets you start your zombie-slaying adventures with your own horse. Don't forget to use the gas to "shine his leathers" and install the saddlebags for more storage. Giddyup!

## Requirements
 - [Telrics Horses V3](https://www.nexusmods.com/7daystodie/mods/7422).
 - 7 Days to Die v3.0 and later.

## Manual installation
 1. Download the mod file.
 2. Unzip it anywhere on your hard drive.
 3. Delete all previous versions of this mod.
 3. Copy the **TelricsHorsesV3-StartingHorseFTLR** folder to the 7 Days to Die Mods folder. The default location is **%APPDATA%\7DaysToDie\Mods**.
 4. If you already have another version of this mod in your Mods folder, delete it first.
 
## Important notes
 - Removing the mod is probably safe, but you should always make a backup of your game save.
