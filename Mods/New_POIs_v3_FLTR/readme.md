# Find New POIs in 7 Days To Die 3.0 - version 3.0.1.1

## Overview

The **Find New POIs in 7 Days To Die 3.0** mod lets you find easily the new POIs added in 7 Days To Die 3.0. Their name appears in orange and is followed by an asterisk. You can also use a less flashy version by copying the Localization.csv file from the \Variants\Asterisk only folder to the Config folder.

POIs marked by a double asterisk do not have an official localization, so I provided a temporary one.

The mod is inspired by CautiousPancake's **New POI Highlight** for 7 Days To Die 2.0.

## Requirements
 - 7 Days to Die v3.0.

## Manual installation

 1. Download the mod file.
 2. Unzip it anywhere on your hard drive.
 3. Copy the New_POIs_v3_FLTR folder to the 7 Days to Die Mods folder. The default location is **%APPDATA%\7DaysToDie\Mods**. 
 4. If you are updating the mod, delete the folder with the old version.
